## рҹӨ” What is HACKEDITOR?

> *Ever wished formatting text felt as cool as hacking into a mainframe?*  
> **HACKEDITOR** is a rich text editor wrapped in a **cyberpunk aesthetic** вҖ” glowing neon buttons, pulsing grids, and a dark-mode-only attitude. No deprecated APIs. No boring UIs. Just pure, modern web tech with maximum style. рҹҳҺ

Built with **zero deprecated APIs** вҖ” uses the modern `Selection` & `Range` API instead of the old `document.execCommand`. Because we don't do legacy here. рҹҡҖ

## рҹҡҖ Getting Started

### Prerequisites

Make sure you have these installed:

- рҹ“Ұ **Node.js** `v18+` вҶ’ [Download](https://nodejs.org/)
- рҹ“Ұ **npm** `v9+` (comes with Node)

### Installation

```bash
# 1пёҸвғЈ  Clone the repo
git clone https://github.com/lrishiraj007/hackeditor.git

# 2пёҸвғЈ  Jump into the directory
cd hackeditor

# 3пёҸвғЈ  Install dependencies
npm install

# 4пёҸвғЈ  Fire it up! рҹ”Ҙ
npm run dev
```

Then open **http://localhost:5173/** in your browser and start hacking! рҹ’»

### Build for Production

```bash

npm run build

# рҹ‘Җ Preview the production build
```

## рҹҺҜ Fun Facts

- рҹ§® **0** deprecated API calls вҖ” we're fully modern here
- рҹҺЁ **3** CSS animations running simultaneously (title glow, grid pulse, status dot)
- рҹ“ә **Scanline overlays** on buttons give authentic CRT monitor vibes
- рҹ§Ҡ The editor area uses `React.memo` with `() => true` вҖ” it literally **never** re-renders from React state changes
- рҹ”Ө The title uses **Orbitron** font вҖ” the same font family used in sci-fi movie UIs
- рҹ’Ў **Zero-width spaces** (`\u200B`) are used to enable formatting at cursor position with no selection
- рҹҢҗ The background grid **breathes** вҖ” it pulses opacity over 8 seconds
- вҡЎ Format state is diffed with a `useRef` before hitting `setState` вҖ” no wasted renders
